# surviver

## 소개

- 개발 동기 : 긴급사태 대처 어플
- 주요 기능
1. 재난 상황 발생 시 대처 방법 및 병원이나 대피소 위치 안내
2. 재난 발생을 인지한 사용자가 재난 발생 위치를 사용자에게 알림
